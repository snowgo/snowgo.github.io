XOR(异或)操作
Python中的XOR操作符是“A”。在二进制位的上下文中 ，0^1和1^0将返回1，而0^0和1^1则会返回0

如果用XOR合并两个数的二进制位，那么把结果数与其中某个操作数重新合并即可生成另一个操作数，这是一个很有用的特性
A^B=C
C^B=A
C^A=B
